# Data Analytics Portfolio

A clean, minimalist, and interactive professional portfolio showcasing my expertise in data analytics, business intelligence, and data-driven decision making.

## 🎨 Features

- **Clean & Minimalist Design** - Professional layout focused on content
- **Fully Responsive** - Mobile-friendly across all devices
- **Interactive Elements** - Smooth animations and hover effects
- **Multiple Sections**:
  - Hero section with call-to-action
  - About me with professional summary
  - Skills & tools showcase
  - Featured projects portfolio
  - Contact section with form
  - Social media links

## 🛠️ Tech Stack

- **Frontend**: HTML5, CSS3, JavaScript (Vanilla)
- **Hosting**: GitHub Pages
- **No Dependencies** - Pure vanilla code, fast and lightweight

## 📊 Skills Highlighted

- **Languages & Databases**: Python, SQL, Excel
- **Business Intelligence**: Power BI, Data Visualization
- **Analytics**: Statistical Analysis, Data Modeling, Forecasting

## 🚀 Getting Started

### Local Setup

1. Clone the repository:
```bash
git clone https://github.com/krishanchoudhary01/portfolio.git
cd portfolio
```

2. Open `index.html` in your browser

### Deploy to GitHub Pages

1. Go to repository Settings → Pages
2. Select `main` branch as source
3. Your portfolio will be live at: `https://krishanchoudhary01.github.io/portfolio`

## 📝 Customization

### Update Personal Information

**In `index.html`:**
- Update hero section heading and subtitle
- Add your contact links (GitHub, LinkedIn, Email)
- Customize about section

**In `style.css`:**
- Modify color scheme (CSS variables at the top)
- Adjust font sizes and spacing
- Customize animations

### Add/Edit Projects

**In `script.js`:**
Update the `projects` array with your projects:

```javascript
const projects = [
    {
        title: 'Your Project Title',
        description: 'Your project description',
        tags: ['Tool1', 'Tool2', 'Tool3'],
        github: 'https://github.com/yourrepo',
        demo: 'https://demo-link.com'
    },
    // Add more projects
];
```
